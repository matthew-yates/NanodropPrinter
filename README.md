# Nanodrop Printer

Syringe dispenser for accurately placing nanoliter scale liquid droplets on surfaces.

### XYZ motion hardware

The motion platform is the [OpenBuilds C-Beam Machine](https://openbuildspartstore.com/openbuilds-c-beam-machine/):
